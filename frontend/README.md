# Reva Electronics Frontend

A dark, futuristic React + TypeScript frontend for Reva Electronics, inspired by SilicOcean features but with a unique design.

## Features
- Service highlights
- Product cards
- Category browsing
- RFQ form
- Contact section

## Color Palette
- Ocean: #016064
- Peacock: #022D36

## Getting Started
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
